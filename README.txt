This is a implementation of the famous NinesMenMorris game in cpp language


go to https://en.wikipedia.org/wiki/Nine_Men's_Morris for instructions

input example:

phase 1:

LuciusVorenus: b2
TitusPullo: d2


phase 2:

LuciusVorenus: e3 d3

please compile using:

g++ nine.cpp Board.cpp GameController.cpp Node.cpp -o game
